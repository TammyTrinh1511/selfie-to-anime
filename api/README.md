## ⚙️ Setup Instructions

### 1. Clone the project

```bash
git clone <your-repo-url>
cd api
source venv/bin/activate  # On macOS/Linux
# or
venv\Scripts\activate     # On Windows
pip install -r requirements.txt
uvicorn index:app --reload
